== Downloading (cloning):
    1. Download and unzip from https://gitea.stockings.mlntcandy.com/netask/modern/archive/master.zip
    2. Create an .ungit/refs file (UTF-8), which will contain same info as https://gitea.stockings.mlntcandy.com/netask/modern.git/info/refs plaintext

== Checking for updates:
    1. Checking remote refs and comparing to local refs from .ungit/refs file
    2. If not similar, download the zip.
    3. Delete mods and config
    4. Uncompress the zip into the game folder (with replace)
    5. Update .ungit/refs to latest refs
